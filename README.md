# Custom Programming Language Project
# Developed By Christian Tampus
## Description:
### A Simple Custom Programming Language That Has The Basic Fundamental Functionality Of A Typical Programming Language
### The Features It Has Contains The Following:
### 1. Single Line Comments
### 2. Variables (INTEGER, DECIMAL, CHARACTER, BOOLEAN, STRING)
### 3. Constants (INTEGER, DECIMAL, CHARACTER, BOOLEAN, STRING)
### 4. Standard I/O (Input & Output)
### 5. Basic Arithmetic Calculated From Left To Right Without Precedence (Addition, Subtraction, Multiplication, Division)
### 6. Control Flow (If, ELSE IF, ELSE)
### 7. Arrays (INTEGER, DECIMAL, CHARACTER, BOOLEAN, STRING)
### 8. Loops (FOR & WHILE)
### 9. Functions w/o Recursion Functionality (VOID, INTEGER, DECIMAL, CHARACTER, BOOLEAN, STRING)
